Core Methods

Methods are blocks of code that perform specific tasks and can be executed when called upon. They promote code reusability and organization.

Defining Methods

In C#, methods are defined within classes. Here's an example of a simple method that adds two integers:

public int Add(int a, int b)
{
    return a + b;
}

Parameters and Return Types

Methods can have parameters to receive input values and can return a value. If a method does not return a value, it is declared with the void keyword.

String Methods

C# provides a variety of built-in methods for string manipulation, such as:

  • Substring(): Extracts a portion of a string.
  • IndexOf(): Returns the index of the first occurrence of a substring.
  • Replace(): Replaces all occurrences of a specified substring with another substring.
  • Split(): Splits a string into an array of substrings based on a specified delimiter.

DateTime and TimeSpan

The DateTime structure is used to work with dates and times, while TimeSpan represents a time interval. These are essential for performing operations involving dates and times in C#.

Mastering methods and their applications is crucial for effective programming in C#.
