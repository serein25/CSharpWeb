Variables & Data Types

In C#, variables are used to store data values. Each variable must be declared with a specific data type, which determines the kind of data it can hold.

Common Data Types

  • int: Represents a 32-bit integer. Example: int age = 30;
  • double: Represents a double-precision floating-point number. Example: double temperature = 36.6;
  • bool: Represents a Boolean value (true or false). Example: bool isActive = true;
  • char: Represents a single Unicode character. Example: char grade = 'A';
  • string: Represents a sequence of characters. Example: string name = "John";

Implicit Typing with var

C# allows implicit typing of variables using the var keyword. The compiler determines the type based on the assigned value.
