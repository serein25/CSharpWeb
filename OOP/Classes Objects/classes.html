Classes & Objects

This section covers the fundamentals of classes and objects in C#. A class is a blueprint that encapsulates data (fields) and behavior (methods), while an object is an instance of that class.

Classes

A class defines a custom type. For example:

public class Car {
    public string Make;
    public string Model;
    
    public void Drive() {
        Console.WriteLine("Driving...");
    }
}

Objects

An object is an instance created from a class. For example:

Car myCar = new Car();
myCar.Make = "Toyota";

This basic setup is the core of object-oriented programming in C#.
