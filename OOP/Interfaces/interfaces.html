Interfaces

Interfaces define a contract that classes can implement, ensuring they provide specific methods and properties without dictating how they are implemented.

Defining an Interface

For example:

public interface ILogger {
    void Log(string message);
}

Implementing an Interface

A class that implements an interface must provide implementations for all its members:

public class ConsoleLogger : ILogger {
    public void Log(string message) {
        Console.WriteLine(message);
    }
}

Interfaces enable loose coupling and enhance the testability and scalability of your code.
