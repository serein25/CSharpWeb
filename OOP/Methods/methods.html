Methods

This section explains how methods work in C#, including instance methods, static methods, and constructors.

Instance Methods

Instance methods operate on a specific instance of a class:

public class Calculator {
    public int Add(int a, int b) {
        return a + b;
    }
}

Static Methods

Static methods belong to the class itself and can be called without creating an instance:

public class MathUtil {
    public static int Multiply(int a, int b) {
        return a * b;
    }
}

Constructors

Constructors are special methods used to initialize new instances of a class. They can also use the this keyword to reference the current instance:

public class Person {
    public string Name;
    
    public Person(string name) {
        this.Name = name;
    }
}

Methods are essential for encapsulating functionality and promoting code reuse.
