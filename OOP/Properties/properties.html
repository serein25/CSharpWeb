Properties & Fields

This section explains the difference between fields and properties in C#. Fields are variables that hold data, while properties provide controlled access to these fields.

Fields

Fields are declared directly in a class and represent its state:

public class Example {
    public int field;
}

Properties

Properties offer a flexible mechanism to read, write, or compute the values of private fields, often with validation logic:

public class Example {
    private int _property;
    
    public int Property {
        get { return _property; }
        set { _property = value; }
    }
}

This approach helps in encapsulating data and providing a controlled interface to it.
