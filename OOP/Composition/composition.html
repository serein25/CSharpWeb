Composition

Composition is an alternative design strategy to inheritance. It involves building complex types by combining objects of other classes.

Example

Consider a Car composed of an Engine, Transmission, and Wheel:

public class Engine {
    public void Start() {
        Console.WriteLine("Engine started");
    }
}

public class Car {
    private Engine engine;
    
    public Car() {
        engine = new Engine();
    }
    
    public void StartCar() {
        engine.Start();
    }
}

This approach promotes modularity and flexibility by allowing objects to be composed of other objects.
