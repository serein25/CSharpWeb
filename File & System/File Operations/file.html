File Operations

This section covers various file operations in C#, including working with the File, FileInfo, FileStream, StreamReader, and StreamWriter classes.

File

The File class provides static methods for creating, copying, deleting, moving, and opening files. For example:

using System.IO;

File.WriteAllText("example.txt", "Hello, world!");
string content = File.ReadAllText("example.txt");
Console.WriteLine(content);

FileInfo

The FileInfo class provides instance methods for file operations and additional file details. For example:

using System.IO;

FileInfo fileInfo = new FileInfo("example.txt");
if (fileInfo.Exists)
{
    Console.WriteLine($"File Size: {fileInfo.Length} bytes");
}

FileStream

FileStream allows you to read from or write to a file as a stream of bytes. For example:

using System.IO;

using (FileStream fs = new FileStream("example.txt", FileMode.OpenOrCreate))
{
    byte[] info = new System.Text.UTF8Encoding(true).GetBytes("Hello, FileStream!");
    fs.Write(info, 0, info.Length);
}

StreamReader and StreamWriter

StreamReader and StreamWriter provide easy-to-use methods for reading and writing text files:

using System.IO;

// Writing to a file
using (StreamWriter writer = new StreamWriter("example.txt"))
{
    writer.WriteLine("Hello, StreamWriter!");
}

// Reading from a file
using (StreamReader reader = new StreamReader("example.txt"))
{
    string line = reader.ReadToEnd();
    Console.WriteLine(line);
}

These file operation classes help you manage files efficiently in C#.
