Directory Operations

This section covers directory-related operations in C#, including the Directory, DirectoryInfo, and Path classes.

Directory

The Directory class provides static methods for creating, moving, and enumerating through directories. For example:

using System.IO;

string path = "NewFolder";
if (!Directory.Exists(path))
{
    Directory.CreateDirectory(path);
}

DirectoryInfo

The DirectoryInfo class provides instance methods for similar operations with additional properties. For example:

using System.IO;

DirectoryInfo dirInfo = new DirectoryInfo("NewFolder");
if (!dirInfo.Exists)
{
    dirInfo.Create();
}

Path

The Path class provides methods to work with file and directory path information. For example:

using System.IO;

string fileName = "example.txt";
string fullPath = Path.Combine("NewFolder", fileName);
Console.WriteLine(fullPath);

These tools simplify working with directories and file paths, making it easier to manage file systems in C#.
