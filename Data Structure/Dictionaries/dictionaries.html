Dictionaries

Dictionaries in C# store key-value pairs and allow fast retrieval of data based on a unique key. They are part of the System.Collections.Generic namespace.

Creating a Dictionary

You can create and populate a dictionary as shown below:

using System.Collections.Generic;

Dictionary<int, string> studentGrades = new Dictionary<int, string>();
studentGrades.Add(1, "A");
studentGrades.Add(2, "B");
studentGrades.Add(3, "A");

Accessing Dictionary Elements

You can iterate over a dictionary to access its keys and values:

foreach (var kvp in studentGrades)
{
    Console.WriteLine($"Student {kvp.Key}: Grade {kvp.Value}");
}

Dictionaries are ideal for scenarios where you need to associate unique keys with specific values.
