Arrays

Arrays in C# are used to store fixed-size sequential collections of elements of the same type. They provide a way to group multiple values into a single variable.

Declaring and Initializing Arrays

You can declare and initialize an array in several ways. For example:

int[] numbers = { 1, 2, 3, 4, 5 };

Alternatively, you can create an array dynamically using the new keyword:

string[] fruits = new string[3];
fruits[0] = "Apple";
fruits[1] = "Banana";
fruits[2] = "Cherry";

Accessing Array Elements

Arrays are zero-indexed, meaning the first element is accessed with index 0. For example:

Console.WriteLine(numbers[0]); // Outputs: 1

Understanding arrays is crucial as they form the basis of working with collections in C#.
