HashSet

A HashSet is a collection that stores unique elements and provides fast lookup performance. It is part of the System.Collections.Generic namespace.

Using a HashSet

Here's an example of how to create and use a HashSet of integers:

using System.Collections.Generic;

HashSet<int> uniqueNumbers = new HashSet<int>();
uniqueNumbers.Add(1);
uniqueNumbers.Add(2);
uniqueNumbers.Add(2); // Duplicate, will not be added
uniqueNumbers.Add(3);

foreach (int number in uniqueNumbers)
{
    Console.WriteLine(number);
}

HashSet is especially useful when you need to ensure that a collection contains no duplicate values.
