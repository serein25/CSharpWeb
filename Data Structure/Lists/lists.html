Lists

Lists in C# are dynamic collections that can grow or shrink in size. They are part of the System.Collections.Generic namespace and are commonly used for managing collections of objects.

Creating and Using Lists

Here's an example of how to create a list of strings, add elements, and iterate over them:

using System.Collections.Generic;

List<string> names = new List<string>();
names.Add("Alice");
names.Add("Bob");
names.Add("Charlie");

foreach (string name in names)
{
    Console.WriteLine(name);
}

Lists provide flexibility and are often preferred over arrays when the number of elements can vary at runtime.
