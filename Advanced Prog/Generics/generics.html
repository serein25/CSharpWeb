Generics

Generics allow you to create type-safe data structures and methods without specifying the exact data type until runtime. They promote code reusability and maintainability.

Generic Classes

A generic class is defined with one or more type parameters. For example:

public class GenericList<T>
{
    private List<T> items = new List<T>();

    public void Add(T item)
    {
        items.Add(item);
    }

    public T GetItem(int index)
    {
        return items[index];
    }
}

Generic Methods

Generic methods allow you to specify type parameters at the method level. For example:

public T Echo<T>(T input)
{
    return input;
}

Using generics helps reduce code duplication and improves type safety throughout your codebase.
