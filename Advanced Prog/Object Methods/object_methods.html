Object Methods

Object methods like GetHashCode and Equals are fundamental to how objects are compared and stored. Overriding these methods correctly is crucial for proper behavior in collections and when checking object equality.

GetHashCode

GetHashCode returns a hash code for the object, which is used in hashing algorithms and data structures such as hash tables. A simple override might look like:

public class Person
{
    public string Name { get; set; }
    public int Age { get; set; }

    public override int GetHashCode()
    {
        return (Name, Age).GetHashCode();
    }
}

Equals

The Equals method determines whether the specified object is equal to the current object. Here's an example override:

public override bool Equals(object obj)
{
    if (obj is Person other)
    {
        return this.Name == other.Name && this.Age == other.Age;
    }
    return false;
}

Overriding both methods in tandem is essential when objects are stored in collections that rely on hashing for equality comparisons.
